# Python-UTN

Ejercicios prácticos sobre la asignatura "Soporte a la Gestión de Datos con Programación Visual"

- Link del repositorio de la materia: https://github.com/frro-soporte/practica-2021

## ¿Qué proyectos hay?

1. Prácticas de la materia (en las carpetas "practicas" correspondientes)
2. Un proyecto Flask en la carpeta practica_07
3. Un proyecto Django en la carpeta practica_07_django
4. Un proyecto kivy en su correspondiente carpeta (un laboratorio de pruebas & un proyecto de un juego)
5. Un proyecto OpenCV en su correspondiente carpeta, sobre detección de rostros y autos en movimiento.
